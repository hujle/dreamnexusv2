# DreamNexus v2.0
An attempt at making improved version of original Yume Nikki Discord Servers list by Dreamnexus.

How to run:

1. Clone the repository.
2. Create .env file from .env.example.
3. Run npm install to install dependencies.
4. Run node server.js to start the server.

Aftet that, web interface can be accessed by going to localhost:port.
